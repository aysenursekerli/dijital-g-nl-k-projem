## 🔤 Özellik 1: Gelişmiş Tipografi

### Açıklama
Metin nesnelerini düzenlerken artık **font stili**, **metin hizalaması** ve **metin rengi** değiştirebilirsiniz!

### Nasıl Kullanılır?

#### Adım 1: Metin Ekleme
- Edit modunda → "Metin Ekle" (type ikonu) tıklayın
- Sayfa üzerine yeni bir metin kutusu eklenir

#### Adım 2: Metin Düzenleme
- Metin kutusuna **çift tıklayın** (double-click)
- Metin editlenebilir hale gelir
- Sağ tarafta **Text Formatting Menüsü** otomatik açılır

#### Adım 3: Özellikleri Değiştirme
Menüde aşağıdaki seçenekler mevcuttur:

**Font Stili (Font Style):**
- 🔤 **Klasik** - Inter (Modern, temiz)
- ✍️ **El Yazısı** - Cursive (Şık, el yazısı tarzı)
- 📜 **Serif** - Georgia (Klasik, formal)
- 💻 **Monospace** - Courier New (Kod tarzı)

**Metin Hizalaması (Text Alignment):**
- ⬅️ **Sola Hizala** - Left align
- ⬆️ **Ortaya Hizala** - Center align
- ➡️ **Sağa Hizala** - Right align

**Metin Rengi (Text Color):**
- 🎨 **Renk Seçici** - Herhangi bir renk seçin
- Seçilen renk hemen metin üzerinde uygulanır

#### Adım 4: Kaydetme
- Metini bitirdikten sonra metin kutusunun dışında klik yapın
- Tüm özellikleri (metin, font, hizalama, renk) otomatik kaydedilir
- Deftere geri döndüğünde aynı özellikler korunur

### Örnek Senaryo
```
1. "Yeni Metin" ekliyorum
2. Çift tıklıyorum → Menü açılıyor
3. Font Stili: "El Yazısı" seçiyorum
4. Hizalama: "Ortaya" seçiyorum
5. Renk: Pembe (#FF69B4) seçiyorum
6. Yazı "Hoşgeldiniz!" yazıyorum
7. Dışında tıklıyorum → Tamamlanıyor!
8. Sayfa kapatıp açtığımda stil korunuyor ✅
```

## 📖 Özellik 2: Spesifik Şablonlar

### Açıklama
Artık sayfa ekleme menüsünde **3 yeni özel şablon** bulunuyor:
- **Kalori Takibi** - Beslenme takibi için
- **Spor Günlüğü** - Egzersiz ve spor aktiviteleri için
- **Bütçe Planlayıcı** - Para ve harcama yönetimi için

### Nasıl Kullanılır?

#### Adım 1: Sayfa Ekleme Menüsünü Açma
- Preview modunda → "Yeni Sayfa Ekle" (file-plus ikonu) tıklayın
- VEYA Sidebar'da → "Yeni Sayfa" (plus ikonu) tıklayın
- Template Modal açılır

#### Adım 2: Şablon Seçimi
Modal'ın **"Spesifik Şablonlar"** bölümünde 3 seçenek görürsünüz:

**🍎 Kalori Takibi**
- Turuncu gradient arka plan (#FFE5B4 → #FFDAB9)
- Beslenme ve sağlık ikonu
- Kalori sayımı, besin takibi için uygun
- Hafif SVG desenleri

**⚽ Spor Günlüğü**
- Mavi gradient arka plan (#B4E5FF → #A8D8FF)
- Spor ve aktivite ikonu
- Antrenman detayları, performans takibi için uygun
- Dinamik görünümlü SVG desenleri

**💰 Bütçe Planlayıcı**
- Yeşil gradient arka plan (#D4FFB4 → #C8E6C9)
- Para ve gider ikonu
- Harcama kategorileri, tasarruf hedefleri için uygun
- Finansal görünümlü SVG desenleri

#### Adım 3: Şablon Seçme
- İstediğiniz şablona tıklayın
- Yeni sayfa otomatik oluşturulur
- Seçilen şablon deseni ve renkleri uygulanır
- Metin ve diğer elementler normalde eklemeye devam edebilirsiniz

### Şablon Özellikleri

Her şablon:
- ✅ Gradient arka plan (profesyonel görünüm)
- ✅ SVG-based dekoratif desenler (hafif, hızlı yükleme)
- ✅ Şeffaf katmanlar (içerik görünürlüğü korunur)
- ✅ Metin, çizim, şekil, resim eklemeye uygun
- ✅ Zoom ve düzenleme işlemleri normal çalışıyor

### Örnek Senaryolar

**Kalori Takibi Örneği:**
```
1. "Yeni Sayfa Ekle" → Kalori Takibi seçiyorum
2. Turuncu arka planlı sayfa oluşturuluyor
3. Tarih ve kalori hedefi metin nesneleriyle ekliyorum
4. Yemekleri listeliyor ve kalori değerleri yazıyorum
5. Kalem ile öğünleri çiziyorum/işaretliyorum
6. Sayfa kaydediliyor ve deftere geri dönüyorum
```

**Spor Günlüğü Örneği:**
```
1. "Yeni Sayfa Ekle" → Spor Günlüğü seçiyorum
2. Mavi arka planlı sayfa oluşturuluyor
3. Antrenman türü, sure ve kalori yakılan info yazıyorum
4. Şekil araçlarıyla grafikler çiziyorum
5. Performans notları ve hedefler ekliyorum
6. Sayfa kaydediliyor
```

**Bütçe Planlayıcı Örneği:**
```
1. "Yeni Sayfa Ekle" → Bütçe Planlayıcı seçiyorum
2. Yeşil arka planlı sayfa oluşturuluyor
3. Gelir ve gider kategorileri metin nesneleriyle ekliyorum
4. Tasnif edilen harcamaları yazıyorum
5. Tasarruf hedefleri ve notları ekliyorum
6. Sayfa kaydediliyor
```

## 🔧 Teknik Bilgiler

### Veri Yapısı - Metin Nesneleri

```javascript
// Metin nesnesinin veri yapısı:
textObject = {
  id: 'media-1234567-890',
  type: 'text',
  content: 'Metin içeriği',
  
  // Yeni OZELLİKLER:
  fontStyle: 'Inter',        // 'cursive' | 'Georgia' | 'Courier New'
  textAlign: 'left',         // 'center' | 'right'
  textColor: '#333333',      // hex renk kodu
  
  // Standart özellikler:
  x: 50,
  y: 50,
  width: 150,
  height: 50,
  rotation: 0,
  zIndex: 10
}
```

### Şablon Sistemı

Şablonlar `pattern-template-*` CSS class'ı kullanılarak uygulanır:

```css
.page.pattern-template-kalori {
  background: linear-gradient(135deg, #FFE5B4 0%, #FFDAB9 100%);
  /* SVG deseni ve katmanlar */
}
```

Veritabanında:
```javascript
pageObject = {
  id: 'pg-1234567-1',
  pattern: 'template-kalori',    // 'template-spor' | 'template-butce'
  bgImage: '',                   // şablonlar bgImage kullanmaz
  media: [...]                   // metin, şekil vb. nesneler
}
```

## 📞 Destek & Sorun Giderme

- ✅ Metin nesnesine **çift tıkladığınızdan** emin olun
- ✅ Edit modunda olduğunuzu kontrol edin
- ✅ Sayfa kaydedip tekrar açmayı deneyin
